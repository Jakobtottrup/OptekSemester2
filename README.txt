1) For at køre denne applikation skal node.js installeres : https://nodejs.org/en/
2) Naviger via kommandoprompt til mappen, der indeholder app.js.
3) Kør kommandoerne "npm install" og "bower install".
4) Start serveren ved at køre kommandoen "node app".
5) Åben browseren og indtast denne adresse "localhost:3000"
